## Hello Raylib

```odin 
package main

import rl "vendor:raylib"

main::proc() {
  rl.InitWindow(1280, 720, "Hello Raylib")
  defer rl.CloseWindow()

  for !rl.WindowShouldClose() {
    rl.BeginDrawing()
    rl.ClearBackground(rl.BLUE)
    rl.EndDrawing()
  }

  }
```

https://www.raylib.com/index.html
